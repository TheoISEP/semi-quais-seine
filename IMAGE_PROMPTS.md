## Conseils pour de meilleurs résultats

### Sur Midjourney
- Utilisez `--ar 16:9` pour les images paysage (hero)
- Utilisez `--ar 4:3` pour les images des cards
- Ajoutez `--v 6` pour utiliser la dernière version
- Ajoutez `--style raw` pour un rendu plus photographique

### Sur DALL-E 3
- Spécifiez "photorealistic" et "professional photography"
- Demandez "high quality, sharp focus"
- Soyez précis sur l'aspect "Seine river in France"

### Paramètres d'export
- **Résolution** : Maximum disponible
- **Format** : JPG
- **Compression** : Passez par TinyPNG après téléchargement

Si vous devez le recréer ou l'adapter :
```
Modern logo design for half marathon event "Le Semi des Quais de Seine", featuring Seine river silhouette, runners silhouettes, bridge icon, navy blue and gold colors, clean professional design, vector style, sports event branding
```

## Optimisation finale

Après avoir téléchargé ou généré vos images :

1. **Redimensionner** :
   - Hero : 1920x1080px
   - Cards : 800x600px
   - Logo : 300x300px

2. **Compresser** avec TinyPNG :
   - https://tinypng.com/
   - Objectif : < 500KB par image
   - Pas de perte visible de qualité

3. **Renommer** exactement comme indiqué :
   - `logo.png`
   - `hero-running.jpg`
   - `parcours.jpg`
   - `ambiance.jpg`
   - `access.jpg`
   - `runners.jpg`

4. **Placer** dans `public/images/`
