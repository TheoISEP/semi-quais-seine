# Images nécessaires pour le site

Placez vos images dans ce dossier avec les noms suivants :

## Obligatoires

### logo.png
- **Description** : Logo de la course "Le Semi des Quais de Seine"
- **Format** : PNG avec fond transparent
- **Dimensions** : 300x300px (ou proportionnel)
- **Utilisation** : Hero section et Footer
- **Note** : Vous avez déjà un logo fourni dans votre brief

### hero-running.jpg
- **Description** : Photo principale pour la section hero
- **Format** : JPG optimisé (< 500KB)
- **Dimensions** : 1920x1080px minimum (format paysage)
- **Sujet** : Coureurs sur les quais de Seine, ou paysage dynamique de la Seine
- **Utilisation** : Arrière-plan du hero (première section)
- **Suggestions** : Photos de foule de coureurs le long de la Seine à Chatou

## Recommandées

### parcours.jpg
- **Description** : Photo du parcours le long de la Seine
- **Format** : JPG optimisé
- **Dimensions** : 800x600px minimum
- **Utilisation** : Card "Un parcours d'exception"

### ambiance.jpg
- **Description** : Photo d'ambiance avec des coureurs
- **Format** : JPG optimisé
- **Dimensions** : 800x600px minimum
- **Utilisation** : Card "Une ambiance unique"

### access.jpg
- **Description** : Photo illustrant l'accessibilité (RER, ville, transport)
- **Format** : JPG optimisé
- **Dimensions** : 800x600px minimum
- **Utilisation** : Card "Accessible depuis Paris"

### runners.jpg
- **Description** : Photo de coureurs de tous niveaux
- **Format** : JPG optimisé
- **Dimensions** : 800x600px minimum
- **Utilisation** : Card "Pour tous les niveaux"

## Sources suggérées

1. **Vos propres photos** d'événements similaires
2. **Banques d'images gratuites** :
   - Unsplash : https://unsplash.com/s/photos/running-paris
   - Pexels : https://www.pexels.com/search/running/
3. **Générateur d'images IA** :
   - Midjourney
   - DALL-E
   - Prompt suggéré : "large group of marathon runners along Seine river in France, aerial view, sunny day, photorealistic"

## Optimisation

Avant d'ajouter vos images, optimisez-les avec :
- **TinyPNG** : https://tinypng.com/
- **Squoosh** : https://squoosh.app/

Objectif : < 500KB par image pour des performances optimales.

## Note importante

Si vous n'avez pas encore toutes les images, le site fonctionnera mais affichera des erreurs d'image dans la console. Ajoutez-les progressivement et le site se mettra à jour automatiquement en développement.
